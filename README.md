# BouncyBalls

## Description
**BouncyBalls** is a fun and interactive CSS animation project that demonstrates multiple balls bouncing inside a container.  
It is built purely with **HTML and CSS**, using **keyframe animations** and **CSS transforms** to create a smooth bouncing effect.  

This project is perfect for beginners or anyone learning **CSS animations**, **timing functions**, and **staggered animations**.

---

## Features

- Multiple balls bouncing with different delays
- Smooth animation using `cubic-bezier`
- Responsive container design
- Pure CSS (no JavaScript required)
- Easy to customize colors, size, and speed
